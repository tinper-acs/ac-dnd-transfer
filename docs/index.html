Ac-dnd-transfer

1. 简介

基于tinper-bee组件库封装开发

在 Tranfer 组件基础上添加拖拽功能

2. 安装

  • 通过ynpm下载使用(现在只支持ynpm进行下载)

ynpm install @yonyou/ac-dnd-tranfer -S

3. 使用

通过ES6的方式进行加载使用

import AcDndTranfer from '@yonyou/ac-dnd-tranfer';//加载组件
